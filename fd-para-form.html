📋 FD → Google Form

Converte automaticamente uma Ficha de Diagnóstico (.docx) em Google Form

1. Configuração (uma vez)

Obtém em Google Cloud Console → Credenciais → Criar ID de cliente OAuth 2.0 (ver instruções)

Como criar o Client ID (uma única vez):

1
Vai a console.cloud.google.com e cria um projeto novo
2
Menu → APIs e Serviços → Biblioteca → pesquisa "Google Forms API" → Ativar
3
APIs e Serviços → Credenciais → Criar credenciais → ID de cliente OAuth 2.0
4
Tipo: Aplicação Web → Nome: "FD para Form" → em "Origens JavaScript autorizadas" adiciona http://localhost:8080
5
Copia o Client ID e cola acima
6
Ecrã de consentimento OAuth → Tipo externo → adiciona o teu email como utilizador de teste

2. Carrega a Ficha de Diagnóstico

Arrasta o ficheiro aqui ou clica para escolher
